Lesson 1: Introduction to Programming
Overview:

Embark on your coding journey with the fundamentals of programming. Learn about variables, data types, and basic control structures.

Programming is a way to instruct the computer to perform various tasks. Understanding variables and data types is crucial as they are the building blocks of any program. Control structures such as loops and conditionals allow us to control the flow of our program.

Challenges to Reinforce Your Learning:
  • Create a simple calculator using variables and basic operations. [LeetCode Problem: Add Two Numbers]
  • Design a program that determines if a number is even or odd. [LeetCode Problem: Palindrome Number]
  • Learn about arrays and use them to solve a problem. [LeetCode Problem: Two Sum]
  • Implement a program that checks for prime numbers. [LeetCode Problem: Count Primes]
Lesson 2: Functions and Methods
Overview:

Dive deeper into programming by learning about functions and methods. Understand how to organize and reuse code effectively.

Functions and methods are reusable pieces of code that can be called with different arguments to perform tasks. They help in breaking down complex problems into simpler, manageable parts. Using functions improves code readability and maintainability.

Challenges to Reinforce Your Learning:
  • Implement a function to calculate the factorial of a number. [LeetCode Problem: Factorial Trailing Zeroes]
  • Create a method to reverse a string without using built-in functions. [LeetCode Problem: Reverse String]
  • Write a function to check if a string is a palindrome. [LeetCode Problem: Valid Palindrome]
  • Develop a function to find the maximum depth of a binary tree. [LeetCode Problem: Maximum Depth of Binary Tree]
Lesson 3: Data Structures
Overview:

Explore the world of data structures. Learn about arrays, lists, and dictionaries, and how to manipulate them effectively.

Data structures are ways of organizing and storing data so that they can be accessed and modified efficiently. Common data structures include arrays, lists, stacks, queues, and dictionaries. Choosing the right data structure can significantly affect the performance of your code.

Challenges to Reinforce Your Learning:
  • Implement a stack using an array and create push/pop operations. [LeetCode Problem: Implement Stack using Queues]
  • Design a program that finds the most frequent element in an array. [LeetCode Problem: Majority Element]
  • Implement a linked list and perform insertion and deletion operations. [LeetCode Problem: Design Linked List]
  • Write a program to check if a binary tree is a valid binary search tree. [LeetCode Problem: Validate Binary Search Tree]
Lesson 4: Sorting Algorithms
Overview:

Understand different sorting algorithms like bubble sort, merge sort, and quick sort. Learn how to implement them in your code.

Sorting algorithms are used to arrange data in a particular order, such as ascending or descending. Each sorting algorithm has its strengths and weaknesses and performs differently based on the data set. Knowing which sorting algorithm to use can help optimize your program's efficiency.

Challenges to Reinforce Your Learning:
  • Implement bubble sort and sort an array. [LeetCode Problem: Sort an Array]
  • Write a program to merge two sorted arrays. [LeetCode Problem: Merge Sorted Array]
  • Implement quick sort and analyze its performance. [LeetCode Problem: Sort Colors]
  • Use merge sort to sort a linked list. [LeetCode Problem: Sort List]
Lesson 5: Recursion
Overview:

Learn about recursion and how to solve problems by breaking them down into smaller subproblems. Practice common recursive techniques.

Recursion is a method where the solution to a problem depends on solutions to smaller instances of the same problem. Some problems are naturally recursive, such as the Fibonacci sequence and the Towers of Hanoi. Mastering recursion involves understanding base cases and recursive cases.

Challenges to Reinforce Your Learning:
  • Write a recursive function to compute the nth Fibonacci number. [LeetCode Problem: Fibonacci Number]
  • Implement a recursive solution to solve the Towers of Hanoi problem. [LeetCode Problem: Tower of Hanoi]
  • Create a recursive function to generate all permutations of a string. [LeetCode Problem: Permutations]
  • Develop a recursive solution to find all subsets of a set. [LeetCode Problem: Subsets]
Lesson 6: Dynamic Programming
Overview:

Discover dynamic programming and how it can be used to solve complex problems by breaking them down into simpler subproblems.

Dynamic programming is a method for solving complex problems by breaking them down into simpler subproblems. It involves solving each subproblem just once and storing their solutions. Common applications of dynamic programming include finding the shortest path and calculating minimum edit distance.

Challenges to Reinforce Your Learning:
  • Solve the knapsack problem using dynamic programming. [LeetCode Problem: Knapsack Problem]
  • Write a program to find the longest common subsequence in a string. [LeetCode Problem: Longest Common Subsequence]
